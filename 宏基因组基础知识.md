# 1.illumina 基本测序方法
主要是分为三个步骤，即**建库**、**cluster（富集）**、**测序**
## 1.1. 建库
- 就是DNA片段的一个集合，将测序片段打碎后形成文库
- 一般1000bp以下成为短DNA文库，反之为长文库，文库的长度指的是打碎后片段长度的峰值
- 打碎后还要进过以下几个步骤：
  1. 加上A碱基进行末端修复（形成粘性末端，方便后面添加index等操作）
  2. 连接测序引物
  3. 连接index
  4. 连接adapter
